Cub

Work ahead of your AI agents, then let them run.

Cub helps developers who are already running AI coding CLIs (Claude Code, Codex, Gemini) in autonomous mode and want more structure.

Install: curl -LsSf https://lavallee.github.io/cub/install.sh | sh

What it does:

  • Prep — Turn ideas into structured, agent-sized tasks through a guided interview
  • Run — Execute tasks autonomously with dependency ordering, retries, and logging
  • Model routing — Route simple tasks to fast models, complex work to capable ones
  • Multi-harness — Works with Claude Code, Codex, Gemini, and OpenCode
