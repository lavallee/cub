# DOCX Skill Documentation

This skill enables Claude to work with Word documents (.docx files).

## Key Workflows

**For Reading Content**: Use pandoc for text extraction or unpack files for raw XML access.

**For Creating Documents**: Follow the docx-js workflow.

**For Editing Existing Documents**: Use the Python-based Document library.
